# zed-printf

A [Zed](https://zed.dev) extension for syntax highlighting printf format strings.

Uses [tree-sitter-printf](https://github.com/pstuifzand/tree-sitter-printf) grammar.

## Highlights

- `%d`, `%s`, `%f`, etc. â€” format specifiers
- Flags, width, precision, and size modifiers
- `%%` escape sequences

## Installation

Search for "printf" in Zed's extension marketplace, or install from source:

1. Clone this repo
2. Open Zed and run `zed: install dev extension`
3. Select the cloned directory
